# Dotfiles

## Installation

### Prerequisites

- [chezmoi](https://www.chezmoi.io/)

#### Ubuntu

```sh
sudo snap install chezmoi --classic
```

#### Windows

```pwsh
winget install twpayne.chezmoi
```

- [Git](https://www.git-scm.com/)

### Init

```sh
chezmoi init git@github.com:yellow-black/dotfiles.git
```
